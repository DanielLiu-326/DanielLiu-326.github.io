确定wan和lan与物理网卡的对应关系：

我这里设置enp2s0为wan，enp3s0为lan，若不同请自行更换对应命令将相关命令通过服务的方式加入开机启动

添加服务，自动配置网络

命令解释：

  • sudo ip link set enp2s0 promisc on #设置网口混杂
  • sudo ip link set enp3s0 promisc on #设置网口混杂
  • **sudo ip link del host_to_openwrt ** #删除已经存在的macvlan
  • sudo ip link add host_to_openwrt link enp3s0 type macvlan mode bridge #新建 host_to_openwrt 接口（类型macvlan）注意网卡对应关系，这里使用lan口的物理网卡作为 macvlan 的对应接口，此处就相当于创建了一个宿主机连接到openwrt 上的桥梁，以后将有此连接承载宿主机的上网流量。
  • sudo ip addr add 10.0.0.10 dev host_to_openwrt 给这个连接设置一个ip地址
  • sudo ip link set host_to_openwrt up 设置该连接启动
  • sudo route add default gw 10.0.0.1 host_to_openwrt 设置路由表，让将宿主机的上网流量走 host_to_openwrt，进而转发给openwrt
  • sudo sh -c ‘echo “nameserver 192.168.6.1” > /etc/resolv.conf’ 设置 dns 为openwrt。
